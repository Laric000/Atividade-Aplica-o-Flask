
## Como Executar Olivia´s Pet

### Clone o repositório:

git clone LINK_DO_REPOSITORIO

### Acesse a pasta do projeto:

cd NOME_DO_PROJETO

### Crie e ative um ambiente virtual:

python -m venv venv

### Cmd:

venv\Scripts\activate

### Execute a aplicação:

flask --app app run --debug

### ou

python app.py

### A aplicação estará disponível em:

http://127.0.0.1:5000
